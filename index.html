Сделайте форму с двумя полями select и кнопкой submit. Второй select изначально не виден.

В первом поле выбора две опции. При выборе второй опции грузится через ajax набор данных для второго select, после загрузки он показывается с обновленными данными. При выборе первой опции в первом select второй select скрывается, выбранное значение в нем обнуляется.

При нажатии кнопки submit данные формы проверяются на заполнение и отправляются через ajax методом POST на сервер. Во время загрузки кнопка submit становится disabled. Форма - bootstrap, обработка - jquery.
